# USAGE
1. cd root_folder
2. run `./install.sh`
3. default shortcut is `ctrl + shift + cmd + 1`  
  
![usage](/images/sort_by_length.gif)
# custom shortcut
1. refer to below keymap
![키맵](/images/keymap.png)

2. open `./install.sh`

3. modify keymap in `./install.sh`, variable of `short_cut`
example) short_cut="^\$@2" # ctrl + shift + cmd + 2

4. run `./install.sh`
5. restart Xcode
